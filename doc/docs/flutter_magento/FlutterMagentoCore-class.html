FlutterMagentoCore

FlutterMagentoCore class

Main singleton class for Flutter Magento library.

This class provides centralized access to all Magento functionality through a singleton pattern. It manages the core services and provides a unified interface for authentication, API services, and cart operations.

Usage

final core = FlutterMagentoCore.instance;

// Initialize the library
await core.initialize(baseUrl: 'https://yourstore.com');

// Access services
final authService = core.authService;
final cartService = core.cartService;
